╔══════════════════════════════════════════════════════════════════════════════╗
║                           USER INTERACTIONS                                  ║
╚══════════════════════════════════════════════════════════════════════════════╝

1. Click Bell Icon
   → Opens/closes notification dropdown

2. Click on a Notification
   → Redirects to orders.php?order_id=[ID]
   → Shows specific order details

3. Click "Clear All" Button
   → Resets notification count to 0
   → Clears all notifications from list
   → Updates last check timestamp

4. Click Outside Dropdown
   → Closes the dropdown
